如何更新博客

2025-02-27 14:16:10#网站搭建
  1. 首先进入hexo的地址，我的地址为 D:/_code/Hexo/mywebsite/
  2. 创建新博客 hexo new "如何更新博客"
  3. 也可以更新博客内容，我的地址为 D:/_code/Hexo/mywebsite/source/_posts
  4. 更新完成之后运行 hexo g 生成文件，然后运行 hexo s 可以本地预览博客，最后运行 hexo d 上传到github上
